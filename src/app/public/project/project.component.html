FangFarm
    Crie um projeto chamado FangFarm ele foi construído usando Spring Boot, Java, REST API e MySQL no backend e no frontend foi utilizado angular CLI.
    O objetivo desse projeto e gerenciar uma fazenda com cadastros básicos de funcionários, veículos e animais.
    Ele pode ser acessado clicando na aba “LOGIN” e clicando no botão entrar.
